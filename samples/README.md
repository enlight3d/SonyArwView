# samples/

Put Sony A7 V / ILCE-7M5 `.ARW` files here to test extraction and thumbnails.

The actual `.ARW` files are **git-ignored** (they are large and often personal).
Drop any A7 V `.ARW` into this folder, then:

```powershell
.\build\src\SonyArwPreviewExtract\Release\SonyArwPreviewExtract.exe --scan-folder samples
```
